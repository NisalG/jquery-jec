This demo shows the default combobox behavior. Select the combobox below and start typing to see the effect.

$('#demo1').jec();

This demo allows you to dynamically disable and enable combobox features.

$('#demo2').jec(); $('#disable').click(function () { if ($(this).val() === 'Disable') { $(this).val('Enable'); $('#demo2').jecOff(); } else { $(this).val('Disable'); $('#demo2').jecOn(); } });

This demo allows you to get and set combobox value.

$('#demo3').jec(); $('#getVal').click(function () { $('#value').val($('#demo3').jecValue()); }); $('#setVal').click(function () { $('#demo3').jecValue($('#value').val()); });

This demo allows you to get and set one of the combobox' preferences (editable option's position).

$('#demo4').jec(); $('#getPref').click(function () { $('#pref').val($('#demo4').jecPref('position')); }); $('#setPref').click(function () { $('#demo4').jecPref('position', Number($('#pref').val())); });
